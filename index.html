Hex Color Codes

We write color codes using the hexadecimal number system, a 16-bit number system that uses two characters (letters or numbers) ranging from 00-FF. Hex color codes consist of three groups of values representing the values red, green and blue. Each color group can produce 256 different colors, and when combined, a total of 16,777,216 colors.

Try creating a red value by moving the red bar to the right. See how the values change as the slider moves. Repeat this with the green and blue sliders to create the colors green and blue.

We create grayscales by placing each slider in the same position on each bar. Purple is a combination of red and blue, and yellow of red and green. Can you find orange and pink?
